# 🐮 **IoT 환경 데이터 수집 & 웹 쇼핑몰 프로젝트** 🐮

🐷 본 프로젝트는 **IoT 데이터 수집**과 **웹 쇼핑몰** 연동을 통해 실시간 환경 데이터를 시각화하고, <br> 사용자 중심의 쇼핑몰 서비스를 구축하는 프로젝트입니다 ! 🌟  
보안 기술을 적용한 실무형 기술 스택을 익히는 것도 핵심 목표입니다 ! 🚀✨

## 🎯 **개발 목적** 🎯

**IoT 데이터 수집**과 **웹 서비스 연동**을 통해 실시간 환경 데이터를 시각화하고, 🎉  
사용자 중심의 쇼핑몰 서비스를 구축하며, 보안 기술을 적용한 실무형 기술 스택을 익히는 것을 목표로 합니다 ! 🌸

<br/><br/><br/>

## 🖥️ **개발 환경** 🖥️

- **운영체제** : Windows 10, 11 💻
- **편집기** : IntelliJ, VSCode ✨

<br/><br/><br/>

## 📋 **요구 조건** 📋

- **사용자 인증 및 권한 부여 시스템** 🔒
- **RESTful API 기반의 클라이언트-서버 연동** 🌐
- **JWT 기반 로그인 및 세션 관리** 🔑
- **웹 쇼핑몰과 실시간 데이터 수집 및 시각화 연동** 📊
- **데이터베이스 관리 및 실시간 처리 최적화** 💾
- **React & Spring Boot 기반의 풀스택 개발 환경 구축** 🛠️

<br/><br/><br/>

## 🚀 **구현 기능** 🚀

1. **사용자와 관리자의 역할에 따른 권한 관리 (JWT 인증)** 🔐  
2. **상품 상세 페이지 구현 및 장바구니 기능** 🛒  
3. **실시간 데이터 수집 및 시각화 기능 (온도, 습도 등)** 🌡️  
4. **주문 내역 관리** 📦  
5. **핸드폰 인증 기능을 통한 실 사용자 인증** 📱  
6. **사용자 인증 및 관리** 👤  
7. **상품 관리 및 쇼핑몰 기능** 🏷️  
8. **Q&A 게시판 기능** ❓  
9. **웹 배포 및 클라우드 설정** ☁️

<br/><br/><br/>

### 🧑🏿‍💻 **개발자** 👩🏿‍💻
- 이 프로젝트의 주요 코드 및 기능은 **KEY, HHW, LLM**님이 개발하셨습니다. 😎  
  KEY, HHW, LLM님의 기여로 **JWT 인증**, **실시간 데이터 수집 및 시각화** 등이 구현되었습니다.
